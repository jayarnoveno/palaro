# Palaro Pass

> Transparent School Sports Event Registration & Fee Disbursement Engine on Stellar.

## Project Description
Palaro Pass eliminates manual cash overhead and administrative delays in school sports competitions across Southeast Asia. School athletic coordinators lock tournament registration fees into Soroban smart contracts, which automatically issue verifiable athlete passes and record game participation on-chain for Basketball, Volleyball, Badminton, and Table Tennis competitions.

## Project Vision
To empower youth sports organizations across emerging markets with zero-friction, transparent financial coordination toolkits on Stellar, turning local athletic meets into verifiable on-chain events.

## Key Features
* **Instant On-Chain Registration:** Sports teams pay registration fees directly in XLM/USDC with immediate transaction confirmation.
* **Automated Entry Verification:** Coordinators verify team readiness and check-in status on-chain.
* **Transparent Pool Audit:** Real-time visibility into collected event registration fees per sports discipline.
